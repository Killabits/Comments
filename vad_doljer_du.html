📡 Konfigurera API-nyckel

Du behöver en gratis YouTube Data API v3-nyckel från Google.

1. Gå till console.cloud.google.com
2. Skapa projekt → APIs & Services → Library
3. Aktivera YouTube Data API v3
4. Credentials → Create API key → kopiera
